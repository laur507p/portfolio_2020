Bates

For our final project of 2nd semester we were given the task to find a danish artist and rebrand their site, create a SoMe strategy and produce video and photo content. We contacted danish graffiti artist Bates, and he agreed to work with us. In this project we used Wordpress REST API to fetch content from a Wordpress backend to create a site with dynamic content.
